Freewrite for Windows

A native Windows port of Freewrite, built for fast local journaling with markdown files, a quiet writing surface, history, export, and fullscreen focus.

Download for Windows View source

Local first

Entries are saved as markdown files in the folder you choose. No account, no cloud sync, no server upload.

Native Windows

Built with WPF and .NET for a real desktop app, not a web wrapper.

Beta release

The first public build is unsigned, so Windows SmartScreen may warn before running it.
